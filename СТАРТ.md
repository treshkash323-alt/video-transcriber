# Video Transcriber — ДЗ-10 · полное руководство

**Путь:** `Projects/ДЗ-10/video-transcriber/`  
**Один терминал:** VS Code Terminal (не несколько `docker compose`!)

---

## 1. Подготовка

1. **Docker Desktop** — кит в трее, Engine running  
2. **`.env`:**
   ```powershell
   cd Projects\ДЗ-10\video-transcriber
   copy .env.example .env
   ```
   ```env
   TRANSCRIPTION_BACKEND=openai
   OPENAI_API_KEY=sk-proj-...
   ```
3. **3 файла** — 30–60 сек, **каждый < 25 МБ**

---

## 2. Запуск (все 4 сервиса)

```powershell
docker compose up --build
```

Ждите:
```
web-1     | Application startup complete
worker-1  | celery@... ready
flower-1  | Visit me at http://0.0.0.0:5555
```

| URL | Что |
|-----|-----|
| http://localhost:8000 | UI |
| http://localhost:5555 | Flower |
| http://localhost:8000/docs | Swagger |

**Скрипт:** `.\scripts\up.ps1`

---

## 3. Загрузка файлов

1. Choose Files — **Ctrl+клик** для 3 файлов  
2. Под полем — размер каждого (красным если >25 МБ)  
3. **Транскрибировать**  
4. Таблица: PENDING → STARTED → SUCCESS  

Можно **по одному** — три раза подряд.

---

## 4. Flower (для ДЗ)

- **Workers** — активный worker  
- **Tasks** — история SUCCESS  

Если :5555 не открывается — перезапуск:
```powershell
Ctrl+C
docker compose up --build
```

Flower теперь **тот же Docker-образ**, что worker (без Connection refused).

---

## 5. Масштаб worker=3

```powershell
Ctrl+C
docker compose up --build --scale worker=3
```

Загрузите 3 файла → в Flower три worker, задачи параллельно.

**Скрипт:** `.\scripts\up-scale.ps1`

---

## 6. Остановка

```powershell
Ctrl+C
docker compose down
```

`uploads/` остаётся на диске.

---

## 7. История задач (SQLite)

После `docker compose down` таблица **не пустая** — задачи в `data/history.db` на диске.

**Скрин для отчёта:**
1. 3 SUCCESS на :8000  
2. `docker compose down`  
3. `docker compose up --build`  
4. Обновить :8000 — **история на месте** (зелёная строка про SQLite)

---

## 8. Частые проблемы

| Симптом | Решение |
|---------|---------|
| 413 / >25 МБ | Файл меньше или `TRANSCRIPTION_BACKEND=local` |
| PENDING | `docker compose logs worker` |
| Flower refused / Inspect method failed | Старый образ или worker ещё стартует | Полный `docker compose up --build`; через 5 сек обновить :5555. Строка `Events of group {task} enabled` = всё OK |
| Port 8000 busy | `docker compose down`, один терминал |
